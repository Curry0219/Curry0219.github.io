the thermocouple thermometer

Posted on 2021-07-09

Well lately I have been busy printing the Progress, and I met a lot more difficulties than I thought due to that the printing machine is not very precise and hence I cannot put all the connections together, some of them might be smaller than I thought or bigger. So now get down to the business, the first software part of the Progress has been down, which is the thermometer part. At beginning, I realized that the temperature- measuring part in Arduino cannot bear the high temperature in the fire, hence I began to find the electronic thermometer with high range and can stay in high temperature for a long time. Combine with the A2 physics I preview, I found that thermometer can perfectly fix my mission.

So I bought the k-type thermocouple and use arduino to code it, make it can transfer the data to the serial monitor in my PC.

#include <SPI.h>
#include “Adafruit_MAX31855.h”

#define MAXCS 4
#define MAXCLK 5

Adafruit_MAX31855 thermocouple(MAXCLK, MAXCS, MAXDO);

void setup() {
Serial.begin(9600);

while (!Serial) delay(1); // wait for Serial on Leonardo/Zero, etc

Serial.println(“MAX31855 test”);
// wait for MAX chip to stabilize
delay(500);
Serial.print(“Initializing sensor…”);
if (!thermocouple.begin()) {
Serial.println(“ERROR.”);
while (1) delay(10);
}
Serial.println(“DONE.”);
}

void loop() {
// basic readout test, just print the current temp
Serial.print(“Internal Temp = “);
Serial.println(thermocouple.readInternal());

double c = thermocouple.readCelsius();
if (isnan(c)) {
Serial.println(“Something wrong with thermocouple!”);
} else {
Serial.print(“C = “);
Serial.println(c);
}
//Serial.print(“F = “);
//Serial.println(thermocouple.readFahrenheit());

delay(1000);
}

Here is all the code, and after I upload the codes, I noticed that the serial monitor shows zero, hence I think there might be something wrong with the connection between the thermocouple and the arduino uno board. Hence in order to make sure that the current can go through the couple properly, I welded all the parts together, and the number is finally showed successfully.
